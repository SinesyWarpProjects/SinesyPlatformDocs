# Sessions and heap memory

This folder reports the **number of user sessions**, related to authenticated users and additional information about the heap memory, in terms of:

* **total heap memory**
* **consumed heap memory**
* **free heap memory**

Finally, a chart reporting the consumption of total/consumed heap memory is reported and refreshed every 2 seconds, when the switcher "Heap Memory Analysis" is pressed: in this way, it is possible to monitor ho the memory is used over time.
